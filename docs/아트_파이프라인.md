# 아트 파이프라인 — 로컬 생성 절차 (2026-07-12)

신규 에셋(몬스터·소품)은 **로컬 diffusion**으로 생성한다. 기존 세트는 유지하고,
**전체 통일 재생성은 백로그**(전투 합격 후 재논의). 주인공·초상은 로컬로 일관 생성 가능함을
검증했으나(아래 §6), 당장은 신규 에셋에만 적용한다.

## 1. 환경
- 실행 파이썬: `D:\ai_factory\contents\modelfit\spike\.venv\Scripts\python.exe` (torch 2.2.1+cu118, diffusers 0.30)
- GPU: GTX 1660 SUPER 6GB — **fp32 강제**(fp16은 검은 NaN). 장당 ~55초, VRAM ~3.8GB, 동시 1장.
- 모델: `tools/models/Inkpunk-Diffusion-v2.ckpt` (SD1.5 파인튠, git 제외). 트리거 `nvinkpunk`.
- IP-Adapter/LCM 등 보조 모델은 ModelFit HF 캐시(`contents/modelfit/spike/hf_cache`) 공용. 추가 다운로드 0.
- :7860 ModelFit 앱과 GPU 공유 → gen_asset이 여유 VRAM 프리체크 후 실행(부족 시 대기·안내).

## 2. 라이선스 (상업 이용 가능)
- **Inkpunk-Diffusion: CreativeML OpenRAIL-M** — 상업 이용 허용(행위제한 조항, 게임아트 무관). 결과물 소유·상업 사용 OK.
- IP-Adapter(Apache-2.0), LCM-LoRA(OpenRAIL++). (Darkest-Diffusion은 비상업 전용이라 **미사용**.)

## 3. 실행 (tools/gen_asset.py)
```
<venv python> tools/gen_asset.py --name <파일명> --model Inkpunk-Diffusion-v2.ckpt \
  --anchor <monster|background|person|경로> --ip-scale 0.65 --seed 7 \
  --sat 0.65 --key-global --prompt "<프롬프트>"
```
출력: `assets/acade/<name>.png`(원본) + `public/art/acade/<name>.webp`(누끼). 여러 `--ip-scale` 주면 `_ip<값>` 변형.

## 4. 톤 기준 (채택값 — 세트 수렴)
- 후처리 tone_match: **`--sat 0.65 --bright 0.86 --contrast 1.2`** + 뼈빛 스플릿톤(기본).
  (채도 크게 낮추고·어둡게·명암 무겁게. sat 0.5는 너무 그레이 → 0.65가 기존 초록 고블린과 조화.)
- 프롬프트 톤 접미(항상 포함): `dark muted palette, heavy shadows, desaturated bone ivory, bold black ink`
- **프롬프트는 77토큰(CLIP) 안**에 — **배경 지시(`plain flat grey background`)를 앞쪽**에 둬서 잘리지 않게. 배경이 평평해야 누끼가 산다.

## 5. 앵커·누끼 규칙
- 앵커는 **같은 부류**를 물린다: monster=`enemy_goblin`, background=`bg_forest`, person=`portrait_hero`(또는 가장 유사한 기존 것). 스타일·정체성 이식 정확도 ↑.
- 누끼: 모서리 flood-fill + **`--key-global`**(배경색 전역 제거 → 잉크튀김에 갇힌 안쪽 헤일로까지 제거). `--nuki-tol` 기본 52.
- **비교샷 필수**: `tools/compare_shot.py --bg public/art/bg_forest.webp --out tools/out/x.png <기존 스프라이트...> <신규>` 로 같은 필드에 나란히 세워 "안 어색한지" 판정.

## 6. 주인공 일관성 (검증됨 — 통일 재생성 시 활용)
`--anchor person`(portrait_hero) + `--ip-scale 0.7~0.9` → 참조와 **동일 인물** 재현 확인
(검은 스파이크 머리·흉터·회색 후드·카타나). 통일 재생성 논의 시 이 방식으로 주인공/초상도 로컬 생성 가능.

## 7. 채택 예 (enemy_archer)
```
gen_asset.py --name enemy_archer --model Inkpunk-Diffusion-v2.ckpt --anchor monster \
  --ip-scale 0.65 --seed 7 --sat 0.65 --key-global \
  --prompt "nvinkpunk, goblin archer drawing a bow, quiver on back, stepping back, facing left, side view, full body, plain flat grey background, dark muted palette, heavy shadows, desaturated bone ivory, bold black ink"
```
