# Package 05 — Duel Foundation (Living Document) v1.0

> 작성일: 2026-07-12 | 선행: Package 03 | 해금: Package 06 | 병렬: Package 04
> 보스전 = Duel Mode의 시스템 기반. "같은 조작을 더 깊게 사용하는 결투."
> **채택 결정(ADR-011 후보)**: 프로토타입에서 검증된 momentum 3상태 구조(대치/공세/수세)를 Duel의 코어로 재설계 채택.

---

# 1. 목적/범위

포함: Duel 상태 머신(공격권), 자세 게이지, 패링/회피 심화 규칙, 거리 관리, Duel 카메라 규칙(원칙만 — 수치는 P12), Duel 진입/종료.
제외: 보스 개별 패턴(P06), 연출 수치(P12).

# 2. 핵심 규칙

1. 항상 1:1. 필드의 속도감 ↔ Duel의 긴장감 리듬 유지.
2. 미니게임 금지 — 필드와 동일한 조작(이동+제스처), 더 깊은 사용.
3. 공격권이 명확하다: 지금 내 턴인지 적 턴인지 화면만 봐도 안다.
4. 죽은 이유는 항상 설명 가능하다.

# 3. Duel 상태 머신 (공격권 시스템)

```
        ┌──────── 대치 (Neutral) ────────┐
        │   서로 간격 조절, 선공 리스크    │
        ▼                                ▼
   내 공세 (Offense)              적 공세 (Defense)
   - 공격권 게이지 시간제          - 적 패턴 시퀀스 진행
   - 모든 획 적중 보정             - 패링(노랑)/회피(빨강)만 유효
   - 화면 가장자리 금빛            - 화면 가장자리 핏빛
        │                                │
        └── 게이지 소진/미스 ──→ 대치 ←── 패턴 종료/패링 성공 ──┘
```
- **공격권 획득**: 패링 성공 / 빨강 회피 후 반격 적중 / 가드 브레이크 / 대치 중 카운터 성립.
- **공격권 상실**: 게이지 소진(기본 6s) / 미스 등급 / 적 슈퍼아머 패턴 발동.
- 상태 표시는 UI 텍스트가 아니라 화면 가장자리 색 + 적 몸 발광 (프로토타입 검증 방식).

# 4. 자세 게이지 (양측)

- 보스 poise: 패링 성공·정타·가드 브레이크로 누적 → 만충 시 **붕괴** = 난무 창(3s, 피해 ×배율, 전 스킬 캔슬 자유).
- 플레이어 poise: 보스 강공 피격·가드 미스 시 누적 → 만충 시 짧은 무방비(1s). 회복은 대치 상태에서만.
- 체력만으로 이기는 보스전 금지 — 자세 붕괴가 주요 승리 경로.

# 5. 패링/회피 심화 (Duel 한정 규칙)

- 노랑 연쇄: 보스 연격은 연쇄 패링 대상 (2~4연). 연쇄 성공 시 poise 보너스 누진 (+60/+70/+85/+100).
- 패링 유효창: 표준 120ms. 난이도 관대 160 / 도전 90 (balance).
- 예고 시간: 노랑 ≥700ms, 빨강 ≥500ms — 보스도 P04 §4 규격 준수. 연쇄 2타부터는 500ms 허용(리듬 학습 전제, 첫 타는 항상 700+).
- 빨강 회피 성공 → 900ms 반격 창 (카운터 배율 적용).
- 거리 3존: 원거리(보스 접근/원거리 패턴) / 중거리(간격 싸움·발도술 존) / 근거리(연격 존). 존별 보스 패턴 풀 분리 (P06).

# 6. Duel 진입/종료

- 진입: 보스 조우 연출 → InputContext = GameplayDuel (INP 규격: 이동 유지, 제스처 취소) → 경기장 경계 활성(이탈 불가).
- 승리: 보스 HP 0 또는 최종 붕괴 → 피니시 연출(P12) → 보상 선택(P08) → Context 복귀.
- 패배: 런 종료가 아니라 **스테이지 시작 체크포인트 부활** (P09) — 보스 재도전 동선 최단화 (Difficulty Bible: 재도전이 먼저 떠오르게).

# 7. Duel 카메라 원칙 (수치는 P12)

- 두 캐릭터 중점 추적 + 간격 비례 줌.
- 공격권 전환 순간 짧은 강조(펀치인).
- 과도한 흔들림 금지 — 패링 성공 순간에만 강한 임팩트.

# 10. Acceptance Test

□ 공격권 전이 매트릭스 전수 (획득 4경로 × 상실 3경로)
□ 공격권 상태가 항상 시각 규칙과 일치 (가장자리 색 이벤트 검증)
□ 연쇄 패링 4연 성공/중간 실패 각 케이스
□ 첫 타 노랑 700ms 미만 데이터 0건 (검증 규칙)
□ 난무 창: 진입/종료/피해 배율 정확
□ 패배→체크포인트→재도전 동선 무결
□ 리플레이 결정론
□ 실기: "긴장→극복" 체감 테스트 (보스 더미로 3연속 결투, 재도전 의욕 질문)
