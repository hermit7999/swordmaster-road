# Sword Language Bible v0.1 (초안 — 사용자 검토 필요)

> 작성일: 2026-07-12
> 상위 문서: 통합 마스터 문서 v1.1 (ADR-010: 벡터 기반, 8방향 양자화)
> 이 문서는 출시 버전의 모든 검술을 정의한다. 제스처 패턴, 역할, 진화, 연계가 여기서 확정되어야 Package 02(Skill Execution)를 시작할 수 있다.
> **⚠ 초안 상태**: 검술 명칭·패턴·역할은 제안이며, 사용자 판정 후 확정한다.

# 1. 설계 원칙

1. **한 검술 = 한 역할.** 모든 검술은 명확한 사용 이유가 있어야 한다 (역할이 겹치면 하나를 삭제).
2. **제스처는 검의 움직임을 닮는다.** 화면에 긋는 선 = 검이 지나가는 궤적. 직관이 곧 튜토리얼이다.
3. **패턴 충돌 최소화.** 8방향 시퀀스가 서로 명확히 구분되어야 오인식률 3% 이하 달성 가능.
4. **진화는 수치가 아니라 플레이 방식을 바꾼다** (Master Bible Rule 6).
5. **적이 검술을 가르친다.** 각 검술은 그것을 요구하는 적과 짝을 이룬다 (ADR-005).

## 8방향 표기
```
NW  N  NE
 W  ·  E
SW  S  SE
```
(우수검류 기준. 좌수검류는 좌우 미러 — E↔W, NE↔NW, SE↔SW)

## 2.1 횡베기 (skill_slash_h)
- **입력**: `E` 단일. 최소 길이 0.25 (정규화 좌표), 직선성 요구.
- **역할**: 리듬의 기본 박자. 전방 가로 부채꼴 범위.
- **짝 적**: 일반 병사 (기본 적 전체).
- **진화**:
  - Lv1 기본 → Lv2 범위 +20% → Lv3 이연격(한 획에 잔상 2히트) → Lv4 관통(직선상 다수 절단) → Lv5 검압(베기 끝에서 짧은 충격파)
- **연계**: 모든 검술로 캔슬 가능한 시작점.

## 2.2 내려베기 (skill_slash_v)
- **입력**: `S` 단일.
- **역할**: 가드 브레이크. 방패·경직 유발 특화, 발동 약간 느림(리스크).
- **짝 적**: 방패병 (정면 공격 무효 → 내려베기로만 방패 파괴).
- **진화**: Lv2 가드 피해 +30% → Lv3 착지 충격(소범위) → Lv4 방어구 관통 → Lv5 대지 가르기(전방 지면 충격파)
- **연계**: 올려베기와 왕복 (낙뢰참 재료).

## 2.3 올려베기 (skill_slash_up)
- **입력**: `N` 단일.
- **역할**: 자세 게이지 피해 특화 → 그로기 메이커. 공중 띄우기.
- **짝 적**: 하단 자세 적, 돌진 적 (카운터로 띄움).
- **진화**: Lv2 자세 피해 +30% → Lv3 띄운 적 추가타 창 → Lv4 대공 판정 확대 → Lv5 승룡 잔상(수직 검기)
- **연계**: 내려베기로 이어 낙뢰참.

## 2.4 사선베기 (skill_slash_diag)
- **입력**: `SE` 단일 (미러: SW도 동일 스킬로 취급할지 → Open Question).
- **역할**: 최속 발동·최저 기력. 콤보 사이를 잇는 접착제. 단독 화력은 낮음.
- **짝 적**: 빠른 검객 (속도 대결).
- **진화**: Lv2 발동속도 +15% → Lv3 연속 사선 2연격 → Lv4 콤보 유지시간 연장 → Lv5 무영검(잔상 분신 1히트)
- **연계**: 연풍참 재료 (횡베기→사선베기).

## 2.5 발도술 (skill_iaido)
- **입력**: `W → E` (짧게 당긴 후 길게 긋기 — 납도→발도).
- **역할**: 간격 싸움의 답. 전방 대시 + 일격. 적 공격 예고에 맞추면 카운터 배율.
- **짝 적**: 검객 (Duel 감각 훈련), 궁수 (거리 좁히기).
- **진화**: Lv2 대시 거리 +30% → Lv3 카운터 배율 ×1.5 → Lv4 발도 중 무적 프레임 → Lv5 신속(화면 절반 관통 이동)
- **연계**: 퍼펙트 발도술 → 오의 일섬 트리거.

## 2.6 역베기 (skill_slash_back)
- **입력**: `W` 단일.
- **역할**: 횡베기의 반환점. E↔W 왕복으로 최고 DPS 연타 리듬. 뒤쪽 판정 보너스(배후 적 견제).
- **짝 적**: 쌍검 적 (리듬 유지 훈련), 협공 상황.
- **진화**: Lv2 왕복 연타 시 피해 누적 +10%/타 → Lv3 후방 판정 확대 → Lv4 왕복 3회마다 자동 검압 → Lv5 팔방베기(왕복이 전방위로)
- **연계**: 횡베기↔역베기 왕복 = 오의 십자참 재료 아님(연무 재료).

## 2.7 회전베기 (skill_spin)
- **입력**: `E→S→W→N` 시계방향 원 (반시계 허용 여부 → Open Question). Corner 3개 이상, 폐곡선 근사.
- **역할**: 포위 해법. 360° 전방위, 높은 기력 비용, 후딜 큼(리스크·리워드).
- **짝 적**: 다수 잡몹 무리, 포위 상황.
- **진화**: Lv2 반경 +25% → Lv3 회전 중 이동 가능 → Lv4 2회전(2히트) → Lv5 검풍(회전 후 도넛형 검기 방출)
- **연계**: 오의 원무 재료.

## 2.8 패링 (skill_parry)
- **입력**: `SE → NE` (✓ 체크 모양 — 받아서 흘려올리는 궤적). 적 공격 적중 직전 ±패링 유효창(balance)에 입력. **[확정 2026-07-12]** ✓ 제스처 유지.
- **✓ 유지에 따른 설계 조건 (필수)**:
  1. **예고 시간 보장**: 패링 가능 공격의 예고(텔레그래프)는 최소 "반응 250ms + ✓ 작성 ~300ms + 유효창" 이상으로 설계. 보스 패턴 설계(Package 05~06)의 제약 조건으로 등재.
  2. **판정 시점 = 긋기 완료 순간** 으로 고정 (근육 기억의 기준점 통일).
  3. **관대한 Corner 인식**: 패링 한정으로 Corner 각도 허용치 완화(다급한 흘림 V 구제). 수치는 balance.
  4. **사선베기 오발 방지**: 패링 유효창 내에서는 `SE` 단독 입력을 사선베기로 실행하지 않고 패링 시도(실패)로 처리 — 방어 시도가 자폭이 되지 않게.
  5. **Acceptance 추가**: "적 예고 → ✓ 입력" 시나리오 실측, 성공률 90%+ (숙련자 기준).
- **역할**: 최고 난이도·최고 보상. 성공 = 기력 회복 + 적 자세 큰 피해 + 반격 창. 실패 = 위치 상실(피해 아님 — Combat Bible §8).
- **짝 적**: 방패병(타이밍 학습), 모든 보스.
- **진화**: Lv2 유효창 +20% → Lv3 성공 시 검기 게이지 +15% → Lv4 연속 패링 보너스 → Lv5 완전 받아치기(성공 시 자동 반격 1타)
- **연계**: 패링 성공 → 어떤 검술로도 캔슬 반격.

# 3. 오의(奧義) 5종

오의 = 검술 연계로 발동하는 필살기. 별도 제스처가 아니라 **정확한 순서 + 시간창 + 등급 조건**으로 성립 (연계창 기본 1.5s, balance로 조정).

| # | 오의 | 성립 조건 | 효과 |
|---|---|---|---|
| 1 | 연풍참(連風斬) | 횡베기 → 사선베기 (굿 이상) | 전방 돌풍 2연격 + 넉백. 최초로 배우는 오의 (Stage 3) |
| 2 | 낙뢰참(落雷斬) | 올려베기 → 내려베기 (그레이트 이상) | 띄운 적을 내려꽂음. 단일 대상 최대 화력 |
| 3 | 십자참(十字斬) | 횡베기 → 내려베기 (그레이트 이상) | 십자 참격. 가드 무시 |
| 4 | 원무(圓舞) | 회전베기 → 사선베기 (굿 이상) | 회전 후 급가속 찌르기. 무리 속 정예 암살 |
| 5 | 일섬(一閃) | 발도술 퍼펙트 → 즉시 발도술 재입력 | 화면을 가르는 섬광. 최고 오의, 검성의 상징 (Stage 4~5 해금) |

- 오의 성립 시: 슬로우 연출 + 전용 사운드 + 마나/검기 보너스 (Game Feel Bible §7 — 실력과 보상 연결).
- 오의는 로그라이크 선택지로 강화 가능 (예: "연풍참에 화상 부여").

# 4. 검기(劍氣) 3종

검기 = 정타로 게이지 충전(kiPerHit), 만충 시 발동 가능한 원거리 참격. 피격 시 게이지 절반 감소.

| # | 검기 | 발동 | 효과 |
|---|---|---|---|
| 1 | 직선 검기(貫) | 만충 + 퍼펙트 횡베기/발도술 | 화면을 가로지르는 관통 참격. 일직선 다수 절단, 방패 무시 |
| 2 | 확산 검기(扇) | 만충 + 퍼펙트 회전베기 | 전방 부채꼴 3갈래 참격. 무리 제거 |
| 3 | 폭렬 검기(爆) | 만충 + 퍼펙트 내려베기 | 자기 중심 원형 방출. 포위 탈출 + 넉백 |

- 발동한 검술의 종류가 검기 형태를 결정 = "어떤 검술로 만충을 터뜨릴까"라는 선택 창출 (Decision Design Bible §1).

# 5. 검파(劍派) 3종과 검술의 관계

| 검파 | 플레이 스타일 | 강화 대상 | 시작 보너스 |
|---|---|---|---|
| 발도류(拔刀流) | 카운터·일격필살. 기다렸다 벤다 | 발도술·패링·일섬 | 발도술 Lv2 시작 |
| 정검류(正劍流) | 정면 대결·중후. 받아내고 부순다 | 내려베기·패링·폭렬 검기 | 패링 유효창 +15% |
| 쾌검류(快劍流) | 속도·연계. 쉬지 않고 벤다 | 사선베기·역베기·연풍참 | 콤보 유지시간 +25% |

- 검파는 검술을 배타 소유하지 않는다 — 전 검술 사용 가능, **강화 방향만 다름** (빌드 다양성).
- 검파 전용 로그라이크 선택지가 존재 (예: 발도류만 "일섬 쿨다운 제거" 등장 가능).

# 6. 제스처 충돌 매트릭스 (오인식 검증 대상)

| 패턴 A | 패턴 B | 충돌 위험 | 구분 근거 |
|---|---|---|---|
| 횡베기 `E` | 발도술 `W→E` | 중 | 발도술은 W 세그먼트 선행 + Corner 1개. 짧은 W가 노이즈로 제거되면 횡베기로 오인 → Noise Filter 임계 주의 |
| 사선베기 `SE` | 패링 `SE→NE` | 중 | 패링은 Corner 1개(V꼴). 사선베기 후 손을 튕겨 올리는 습관 입력 주의 → 패링의 NE 최소 길이 요구 |
| 횡베기 `E` | 사선베기 `SE` | 저 | 방향 오차 ±20° 내에서 E(0°)와 SE(-45°)는 경계 22.5° 분리 — 판정 경계 명확 |
| 올려베기 `N` | 역베기 `W` | 저 | 90° 분리 |
| 회전베기 (원) | 기타 전부 | 저 | Corner 3+와 폐곡선은 유일 |

**Acceptance 추가 항목**: 위 충돌 쌍 각 500회 입력 → 오인식 3% 이하 (Gesture Acceptance Test Spec에 추가).
