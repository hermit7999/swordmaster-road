# 4. 보상 타이밍

3\~10초 - 타격감

30초 - 작은 보상

1분 - 성장 선택

5분 - 보스 보상

30분 - 영구 성장

보상 공백이 길어지지 않도록 설계한다.

------------------------------------------------------------------------

# 5. 반복 플레이 이유

플레이어는 다시 시작하는 이유가 최소 하나 이상 있어야 한다.

-   다른 검술
-   다른 검파
-   다른 빌드
-   더 높은 난이도
-   더 높은 숙련도

------------------------------------------------------------------------

# 6. Core Loop 체크리스트

□ 전투가 재미있는가?

□ 보상이 빠른가?

□ 성장이 체감되는가?

□ 보스가 기대되는가?

□ 다음 플레이 이유가 있는가?

YES 5개가 목표.

------------------------------------------------------------------------

# 7. Core Loop를 깨는 요소

-   보상 없는 긴 전투
-   HP만 많은 적
-   성장 체감 부족
-   너무 긴 이동
-   반복되는 동일 패턴

발견 즉시 수정 대상.
