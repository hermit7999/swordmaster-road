# 1. 이벤트 프로파일 시스템 (연출의 데이터화)

전투 로직(P03)은 profile ID만 발행 — 연출 수치는 전부 `event_profiles.json`:

```json
{
  "profile_id": "hit_medium",
  "hit_stop_ms": 70,
  "shake": { "amp": 4, "ms": 90 },
  "flash": { "target": "enemy", "ms": 50 },
  "sfx": "sfx_slash_hit_02",
  "vfx": "vfx_ink_splash_m",
  "haptic": "light"
}
```

기본 프로파일 세트 (초기값):

| profile | hit_stop | shake | 용도 |
|---|---|---|---|
| hit_light | 40ms | 2/60ms | 잡몹 일반타 |
| hit_medium | 70ms | 4/90ms | 정타·다수 절단 |
| hit_heavy | 110ms | 6/120ms | 오의·가드브레이크 |
| parry_perfect | 140ms + 프레임 정지 | 5/100ms + 플래시 | 패링 성공 |
| finish_wave | slow 0.3×760ms + 줌 | — | 무리 피니시 (프로토타입 검증값) |
| finish_boss | slow 0.25×1200ms + 줌 + 세로 갈라짐 | — | 보스 피니시 |

# 2. 카메라 규칙

| 상황 | 동작 |
|---|---|
| 필드 | 플레이어 추종 + 진행 방향 룩어헤드(+1.5unit), 부드러운 감쇠 |
| 필드 전투 | 미세 줌인(1.05) — 적 전멸 시 복귀 |
| 무리 피니시 | 마지막 적 포커스 줌(1.3) + 슬로우 |
| Duel | 두 캐릭터 중점, 간격 비례 줌(0.9~1.2), 경기장 경계 클램프 |
| 공격권 전환 | 펀치인 80ms |
| 보스 등장 | 보스 풀샷 1.2s → 복귀 (스킵 가능, 재도전 시 자동 스킵) |
| 흔들림 | 옵션 강도 배율(접근성). 동시 shake는 최대값 1개만 (중첩 금지) |

# 3. 검 궤적 & 판정 피드백 (게임의 얼굴)

- 플레이어 획 = 잉크 라인 실시간 렌더 (긋는 중 표시, 판정 후 등급색으로 펄스: 퍼펙트 금/그레이트 은/굿 백/배드 회).
- 오인식 방지 피드백: 미스 시 궤적이 붉게 흩어짐 + 실패 사유 아이콘 (개발 모드: 상세).
- 적 피격: 잉크 스플래시 + 넉백 + 흰 플래시 50ms. HP만 감소 금지 (Game Feel §4).
- 검기: 화면 가로지르는 참격 라인 (프로토타입 acProjectiles 검증).

# 4. 오디오

- 필수 SFX 세트: 휘두름(획 종류별 3변형), 명중(재질 2종×3변형), 가드/막힘, 패링(전용 금속음 — 가장 공들일 소리), 그로기 균열, 피니시, 오의 전용 5종, UI.
- 같은 소리 연속 반복 금지 — 변형 라운드로빈 (Game Feel §6).
- BGM: 스테이지 5 + Duel 5 + 타이틀/결과. Duel 진입 시 크로스페이드, Phase 전환 시 강도 상승 레이어.
- 웹 제약: AudioContext는 첫 탭 후 활성 (타이틀 "탭하여 시작" — P10).

# 5. 아트 방향

- 스타일: 수묵(sumi-e) 잉크 + 플랫 채색, 다크 판타지 (프로토타입 아트 파이프라인 자산 승계 — Inkpunk 로컬 생성 파이프라인·톤 매칭 후처리 재사용 가능, `docs/아트_파이프라인.md` 참조).
- 캐릭터: 포즈 스프라이트 기반 (idle/run/slash/slam/hit/dead 등 6~10포즈) — 풀 골격 애니메이션 지양 (1인 규모).
- 배경: 패럴랙스 2~3겹.
- 라이선스 규칙: 상업 사용 가능 모델만 (OpenRAIL-M 등), 모델·라이선스를 아트_파이프라인.md에 기록 (기존 절차 유지).

# 7. Acceptance Test

□ 첫 베기 5요소(즉시 반응/궤적/타격음/적 반응/히트스톱) 동시 발생 (Game Feel §3)
□ 동시 다중 히트 시 shake 중첩 없음·히트스톱 최대 1회
□ 슬로우 중 입력 판정 정상 (시간 스케일이 게임 로직 dt에만)
□ 60FPS: 적 6 + VFX 폭발 상황 (폰 실기)
□ 사운드 변형 라운드로빈 동작
□ 흔들림 0 옵션에서 완전 정지
□ 실기 설문: "베는 맛" 5점 중 4+ (테스터 3인+)
