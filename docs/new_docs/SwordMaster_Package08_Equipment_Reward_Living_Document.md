# 1. 목적/범위

포함: 장비 슬롯/Modifier, 드랍 규칙, 골드 경제, 보상 선택 흐름, 상점(휴식처), reward/item.json.
제외: 성장 선택지 자체(P07), 상점 UI(P10).

# 2. 장비 시스템

- 슬롯 3: **검(武)** / **호구(防)** / **비전(祕)** (장신구 개념).
- 장비 = Modifier 묶음. 공격력 +1000 금지 — 전 옵션이 조건부/스타일형:

| 슬롯 | Modifier 예시 (풀 12+) |
|---|---|
| 검 | 발도술 사거리 +30% / 관통 +1 / 퍼펙트 시 검기 +50% / 오의 위력 +20% & 기본기 -10% |
| 호구 | 회피 후 300ms 이속 +40% / 피격 시 검기 미감소(런당 3회) / poise 피해 -20% |
| 비전 | 패링 성공 시 HP 2 회복 / 콤보 10+ 시 크리 +15% / 그로기 연장 +1s |

- 수치형 옵션 상한: 단일 옵션 위력 기여 ≤ +30% (실력 대체 방지 가드레일).
- 시너지 태그: modifier에 build_tag(발도/패링/콤보/검기) — 도감·선택지와 연동해 빌드 연구 재미(Archetype 3).

# 3. 드랍/획득 규칙

- 잡몹: 골드만 (아이템 파밍 게임 방지).
- 정예/미니보스: 장비 1 확정 (희귀도 추첨).
- 보스: 성장 3택(P07) + 골드 + 수련점. 장비는 상자/상점 경유.
- 희귀도: 일반/정교/걸작/전설. 전설 = 플레이 방식 전환급 (예: "패링이 회피로도 성립").

# 4. 골드 경제

- 획득: 처치, 웨이브 보너스(노데미지 배율 ×1.5 — 실력 보상), 스킵 보상.
- 사용: 휴식처 상점 — 장비 3종 / 소모품 2종 / 재추첨 1회.
- 소모품: 회복약(휴식처 외 사용 불가 — 전투 중 아이템 금지로 긴장 유지), 숫돌(다음 보스전 한정 버프).
- 사망 시 골드 소실 (런 자원). 경제 곡선 목표: 스테이지당 상점 1회, 살 수 있는 것 1~2개 (선택 압박 유지).

# 5. 보상 연출 흐름 (Reward Bible 5단계 매핑)

- L1(0~3초): 타격 피드백 — P03/P12.
- L2(30초): 웨이브 클리어 골드 팝 + 노데미지 배율 표시.
- L3(1~5분): 성장 3택 화면 (P07/P10) — 전투 정지, 여유 있는 선택.
- L4(세션): 결과 화면 — 수련점 정산 + 미해금 미리보기 1개.
- L5(장기): 도감/승급 (P07).

# 6. 데이터 스키마

`items.json`: {item_id, slot(weapon|armor|secret|consumable), rarity, modifiers[{stat, op, value, condition?, build_tag}], price, drop_weight, version}
`reward_tables.json`: {table_id, entries[{type(gold|item|option_roll), weight, range}]}
`economy.json`: {gold_curve[스테이지별 기대 수입], shop_price_mul, reroll_cost}

# 8. Acceptance Test

□ 단일 옵션 위력 기여 ≤30% 검증 규칙 (데이터 린트)
□ 조건부 Modifier 발동/미발동 경계 정확
□ 드랍 분포 1만회 시뮬 = 설계 확률 ±1%
□ 노데미지 배율이 실제 노데미지에서만
□ 잡몹 장비 드랍 0건
□ 경제 시뮬: 5스테이지 표준 플레이 시 상점 구매 1~2개 압박 성립
□ 전 장비: "플레이 방식이 바뀌는가?" 질문 통과 (설계 리뷰)
