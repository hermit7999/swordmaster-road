# 문서 우선순위 기준

문서는 아래 질문으로 평가한다.

1.  이 문서가 없으면 개발이 멈추는가?
2.  프로그래머가 바로 구현할 수 있는가?
3.  플레이 테스트에 직접 영향을 주는가?

YES가 많을수록 우선순위가 높다.
