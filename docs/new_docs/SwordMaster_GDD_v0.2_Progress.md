# SwordMaster GDD v0.2 - 진행 기록 및 다음 설계

# 2026-07-12 설계 결론

## 프로젝트 방향

이 프로젝트는 AAA 액션 게임을 만드는 것이 아니다.

목표는

> **1인 개발자가 완성 가능한, 손으로 검술을 직접 사용하는 액션
> 로그라이크**

이다.

------------------------------------------------------------------------

# Core Vision

플레이어는 버튼으로 스킬을 사용하는 것이 아니라, 자신의 손으로 검술을
직접 시전한다.

------------------------------------------------------------------------

# Core Fun

-   손으로 검술을 사용하는 재미
-   검술을 익히는 재미
-   검객으로 성장하는 재미
-   보스와의 긴장감 있는 결투

------------------------------------------------------------------------

# 게임 구조

필드(80%)

-   왼손 이동
-   오른손 검술
-   빠른 전투
-   5분 플레이

↓

보스(20%)

-   Duel Mode
-   같은 조작
-   더 깊은 검술
-   짧고 강렬한 결투

↓

새 검술 또는 강화 선택

↓

다음 스테이지

------------------------------------------------------------------------

# 현재 결정 사항

-   일반전 : 보스 = 8 : 2
-   플레이 타임 = 5분 단위
-   보스는 미니게임이 아니라 Duel Mode
-   세키로를 복제하지 않는다.
-   세키로의 긴장감만 참고한다.
-   시스템은 1인 개발 가능한 수준으로 유지한다.

------------------------------------------------------------------------

# 새롭게 결정된 방향

게임의 성장은 레벨업이 아니다.

게임의 성장은

"검객으로 성장하는 과정"

이다.

하지만

검사 → 검객 → 검호 → 검성

4단계는 너무 적다.

플레이어는 자주 성장했다는 느낌을 받아야 한다.

------------------------------------------------------------------------

# 새로운 검객 성장 체계 (초안)

1.  견습 검사
2.  초급 검사
3.  중급 검사
4.  상급 검사
5.  검객
6.  숙련 검객
7.  검호
8.  대검호
9.  검왕
10. 검성
11. 검신

※ 이름은 추후 변경 가능

중요한 것은 이름보다 각 단계마다 새로운 검술, 오의, 패시브, 플레이
스타일이 열려야 한다.

------------------------------------------------------------------------

# 성장 방식

성장은 경험치가 아니라

-   검술 숙련
-   보스 격파
-   새로운 오의 습득
-   검술 도전 과제

를 통해 이루어진다.

------------------------------------------------------------------------

# 다음 설계 우선순위

## 1. Sword Progression Bible (최우선)

각 성장 단계에서

-   무엇을 배우는가
-   무엇이 해금되는가
-   어떤 플레이가 달라지는가

를 정의한다.

------------------------------------------------------------------------

## 2. 로그라이크 선택 시스템

보스 처치 후

3개의 선택지 제공

예)

-   발도술 강화
-   회전베기 강화
-   새로운 검술 습득

플레이마다 다른 빌드 형성.
