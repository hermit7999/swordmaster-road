# 수익화 설계 v1.0 (ADR-008 후속)

> 작성일: 2026-07-12 | PC+모바일 동시 플랫폼에 맞춘 수익화 확정안.
> 불변 원칙: 실력으로 강해지고, 콘텐츠에 돈을 쓴다. Pay to Win 절대 금지 (마스터 문서 §11).

---

# 1. 채널별 모델

| 채널 | 모델 | 내용 |
|---|---|---|
| 웹 (데모) | 무료 | S1 전체 (First 30 Minutes 범위) — 마케팅·플레이테스트 채널. 위시리스트/스토어 유도 |
| Steam | 프리미엄 (본편 유료) | 전체 5스테이지 + 이후 무료 업데이트(B티어). DLC(C티어) 별도 판매 |
| 모바일 | 프리미엄 동일 가격 | IAP는 **DLC 언락 + 꾸미기만** — 소모성 IAP/광고/가챠 없음 |

- 가격: 미정 (Open — 유사 장르 조사 후. 기준: 소규모 프리미엄 인디 밴드).
- 한 번 구매 = 전 채널 동일 콘텐츠 (크로스 세이브는 P11 클라우드 이후).

# 2. 판매 가능 목록 (기존 GDD §11 승계·확정)

- DLC: 신규 지역(스테이지 세트)+보스+검파+검술 계통 묶음.
- 꾸미기: 의상/검 외형/검 이펙트 색/피니시 연출/칭호 — **성능 영향 0** (데이터 린트로 강제).

# 3. 판매 금지 (린트 규칙화)

능력치, 검술 위력, 수련점, 골드, 부활, 편의(인벤토리 등 진행 관련) — item/growth 데이터에 `purchasable` 필드 자체를 두지 않음 (구조적 방지).

# 4. 단계별 로드맵

1. **웹 데모 공개** (무료) — 텔레메트리로 재미 검증 + 커뮤니티 시드
2. **Steam 출시** (본편) — 데모는 Steam 데모로도 병행
3. **무료 업데이트 2~3회** (챌린지·보스러시) — 평가·커뮤니티 축적
4. **DLC 1** (검파+1, 보스+2, 검술+3 — Scope Bible)
5. **모바일 출시** (본편 이식 + 그동안의 업데이트 포함)
