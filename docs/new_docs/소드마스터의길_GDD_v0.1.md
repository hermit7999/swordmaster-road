# 소드마스터의 길(Game Design Document v0.1)

## 프로젝트 목표

**1인 개발자가 1년 내 완성 가능한 검술 액션 게임**

핵심은 거대한 오픈월드가 아니라 **'마우스로 검술을 직접 사용하는 재미'**
이다.

------------------------------------------------------------------------

# Core Vision

> 플레이어는 버튼으로 스킬을 사용하는 것이 아니라, 자신의 손으로 검술을
> 직접 시전한다.

------------------------------------------------------------------------

# Core Fun

-   내 손으로 검술을 직접 사용하는 재미
-   점점 검객에서 검성으로 성장하는 재미
-   필드에서는 시원하게 베는 재미
-   보스에서는 긴장감 있는 결투를 하는 재미

------------------------------------------------------------------------

# Player Fantasy

최종 목표는 "검성"이 되는 것이다.

레벨보다 검술의 숙련이 중요하다.

------------------------------------------------------------------------

# 장르

Gesture Sword Action RPG

-   필드 : 아케이드 액션 + 가벼운 RPG
-   보스 : 검술 결투(Duel)

------------------------------------------------------------------------

# 게임 구조

    필드 탐험
        ↓
    몬스터 처치
        ↓
    검술 숙련
        ↓
    보스 발견
        ↓
    Duel Mode 진입
        ↓
    보스 처치
        ↓
    새 검술 습득
        ↓
    다음 지역

------------------------------------------------------------------------

# 일반 전투

목표는 '압도'이다.

-   왼손 : 이동
-   오른손 : 검술 제스처

일반 몬스터는 5\~10초 내 처치.

플레이어는 계속 이동하고, 끊임없이 적을 베며 앞으로 나간다.

------------------------------------------------------------------------

# 보스전

보스전은 별도의 Duel Mode.

하지만 미니게임처럼 완전히 다른 게임이 아니라, 같은 조작을 더 깊게
사용하는 결투이다.

### 특징

-   항상 1:1
-   카메라를 보스 중심으로 유지
-   거리 조절
-   좌우 회피
-   검술 제스처

세키로의 긴장감은 참고하지만, 구현은 훨씬 단순하게 한다.

------------------------------------------------------------------------

# 보스 설계 원칙

세키로처럼 수십 개 패턴을 만들지 않는다.

### 보스 패턴

-   1보스 : 3개
-   2보스 : 4개
-   최종보스 : 최대 6개

패턴은 적지만 조합을 통해 난이도를 만든다.

------------------------------------------------------------------------

# 검술 시스템

초기에는 5\~8개의 핵심 검술만 구현.

예시

-   횡베기
-   발도술
-   회전베기
-   패링
-   일섬

새로운 검술을 무한히 늘리지 않는다.

------------------------------------------------------------------------

# 성장 시스템

레벨보다 검술 성장.

예)

-   기본 베기
-   발도술
-   회전베기
-   패링
-   일섬

필드에서 익히고 보스전에서 시험받는다.

------------------------------------------------------------------------

# 일반전과 보스전 비율

80 : 20

필드는 빠른 템포.

보스는 긴장감 있는 결투.

------------------------------------------------------------------------

# 플레이 세션

1회 플레이

약 5분

    5분 플레이
        ↓
    보스
        ↓
    새 검술
        ↓
    다음 5분

로그라이크 구조를 참고.

------------------------------------------------------------------------

# 개발 철학

## 하지 않을 것

-   거대한 오픈월드
-   NPC 수십 명
-   장비 수백 개
-   복잡한 퀘스트
-   AAA급 AI

## 집중할 것

-   검술
-   제스처
-   손맛
-   검객 판타지
-   보스 결투

------------------------------------------------------------------------

# 구현 우선순위

1.  왼손 이동 + 오른손 검술
2.  필드 전투
3.  Duel Mode
4.  검술 5\~8개
5.  보스 3명
6.  로그라이크 성장
7.  Steam Demo

------------------------------------------------------------------------

# 성공 기준

사용자가

"5분만 해야지."

라고 시작했는데

1시간이 지나 있는 게임.

그리고

"마우스로 검술을 사용하는 게임은 처음이다."

라는 평가를 받는 것이 목표이다.
