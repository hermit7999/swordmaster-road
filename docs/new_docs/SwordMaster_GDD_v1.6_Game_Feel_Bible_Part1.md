# SwordMaster GDD v1.6

# Game Feel Bible (Part 1)

> 이 문서는 게임의 '감각(Game Feel)'을 정의한다. Combat Bible이 전투
> 규칙이라면, Game Feel Bible은 플레이어가 느끼는 감정을 정의한다.

------------------------------------------------------------------------

# 1. 핵심 목표

플레이어가 검을 한 번 그었을 때

"와, 이거 계속 베고 싶다."

라는 느낌을 받아야 한다.

손맛은 그래픽보다 우선한다.

------------------------------------------------------------------------

# 2. Game Feel의 7가지 요소

1.  입력(Input)
2.  반응(Response)
3.  타격감(Hit Feel)
4.  화면(Camera)
5.  사운드(Audio)
6.  이펙트(FX)
7.  보상(Reward)

이 7가지가 동시에 동작해야 한다.

------------------------------------------------------------------------

# 3. 첫 베기 원칙

첫 번째 베기가 게임의 인상을 결정한다.

필수 요소

-   즉시 반응
-   선명한 검 궤적
-   강한 타격음
-   적의 명확한 반응
-   짧은 히트스톱

------------------------------------------------------------------------

# 4. 적 반응 원칙

모든 공격은 적에게 눈에 보이는 변화를 만들어야 한다.

예시

-   뒤로 밀림
-   자세 무너짐
-   방어 자세
-   비틀거림
-   그로기

HP만 감소하는 것은 금지한다.

------------------------------------------------------------------------

# 5. 카메라 원칙

카메라는 플레이어를 방해하지 않는다.

카메라는

-   중요한 순간만 강조
-   평소에는 안정적
-   보스전은 긴장감 증가

과도한 흔들림은 금지한다.

------------------------------------------------------------------------

# 6. 사운드 원칙

사운드는 타격감을 증폭시킨다.

필수 사운드

-   검 휘두름
-   명중
-   패링
-   그로기
-   피니시

같은 소리를 반복하지 않는다.

------------------------------------------------------------------------

# 7. 보상 원칙

플레이어가 잘했을 때만 큰 연출을 준다.

예)

-   완벽한 패링
-   연속 콤보
-   노데미지 처치
-   보스 피니시

실력과 보상이 연결되어야 한다.

------------------------------------------------------------------------

# 8. 실패 원칙

실패는 억울하면 안 된다.

플레이어는

"게임이 잘못됐다."

가 아니라

"내가 조금 늦었다."

라고 느껴야 한다.

------------------------------------------------------------------------

# 9. 금지 사항

-   긴 피격 경직
-   입력 씹힘
-   이유 없는 피격
-   과도한 화면 효과
-   적 체력만 높은 설계

------------------------------------------------------------------------

# 10. Game Feel Checklist

새로운 기능을 넣기 전에 확인한다.

□ 첫 3초가 재미있는가?

□ 첫 베기가 시원한가?

□ 적 반응이 명확한가?

□ 성공했을 때 기분이 좋은가?

□ 실패했을 때 다시 하고 싶은가?
