# SwordMaster GDD v2.4

# Scope Bible (출시 범위 정의)

> 목적
>
> 이 문서는 **무엇을 만들 것인가**가 아니라 **어디까지 만들고 출시할
> 것인가**를 정의한다.
>
> Scope는 품질을 지키기 위한 약속이다.

------------------------------------------------------------------------

# 1. 출시 원칙

출시는 "기능이 많을 때"가 아니라

**핵심 재미가 완성되었을 때** 한다.

------------------------------------------------------------------------

# 2. MVP 출시 범위

## 플레이

-   플레이 시간 : 30\~40분
-   스테이지 : 5개
-   일반전 + Duel Mode

------------------------------------------------------------------------

## 검술

출시 버전

-   기본 검술 : 8개
-   오의 : 5개
-   검기 : 3종

확장팩에서 추가 가능

------------------------------------------------------------------------

## 검파

출시

-   3개

예시

-   발도류
-   정검류
-   쾌검류

업데이트마다 1개씩 추가 가능

------------------------------------------------------------------------

## 보스

출시

-   메인 보스 : 5명
-   미니보스 : 5명

각 보스는 "새로운 시험"을 제공해야 한다.

------------------------------------------------------------------------

## 일반 적

출시

-   10\~12종

원칙

적마다 하나의 학습 목표가 있어야 한다.

------------------------------------------------------------------------

## 성장

출시

-   영구 성장
-   로그라이크 선택
-   검술 진화

추가 성장 시스템은 출시 이후 검토

------------------------------------------------------------------------

# 3. 출시 제외 항목

출시 이후 확장

-   신규 지역
-   신규 검파
-   신규 검술
-   챌린지 모드
-   보스 러시
-   최고 난이도
-   숨겨진 엔딩

------------------------------------------------------------------------

# 4. 콘텐츠 품질 기준

새 콘텐츠를 추가하기 전에 확인

□ 기존 콘텐츠보다 재미있는가?

□ 기존 시스템을 더 깊게 만드는가?

□ 개발 기간 대비 가치가 높은가?

아니라면 기존 콘텐츠를 개선한다.

------------------------------------------------------------------------

# 5. DLC 정책

## DLC 1

-   검파 +1
-   보스 +2
-   검술 +3

## DLC 2

-   신규 지역
-   신규 적
-   신규 보스
-   신규 엔딩

------------------------------------------------------------------------

# 6. Scope Lock

다음 항목은 프로토타입 이후에는 쉽게 변경하지 않는다.

-   조작 방식
-   Core Loop
-   Duel Mode
-   성장 철학
-   플레이 시간 목표

변경 시 ADR에 기록한다.

------------------------------------------------------------------------

# 7. Scope Review

매월 확인

-   콘텐츠가 늘어나기만 하는가?
-   출시 일정이 밀리는가?
-   핵심 재미보다 기능이 늘고 있는가?

YES가 나오면 범위를 줄인다.
