# SwordMaster GDD v2.1

# Player Archetype Bible

> 목적
>
> 이 문서는 플레이어 유형을 정의한다. 새로운 기능은 반드시 하나 이상의
> 플레이어 유형에게 명확한 가치를 제공해야 한다.

------------------------------------------------------------------------

# 핵심 원칙

좋은 게임은 모든 사람을 만족시키려 하지 않는다.

대신 서로 다른 플레이 성향이 각자의 재미를 찾을 수 있도록 설계한다.

------------------------------------------------------------------------

# Archetype 1 : 검술 마스터 (Mastery)

## 목표

내 실력이 늘어나는 것이 즐겁다.

## 좋아하는 요소

-   패링
-   Duel Mode
-   노데미지 클리어
-   빠른 클리어
-   최고 난이도

## 싫어하는 요소

-   운빨
-   Pay to Win
-   스탯만 높은 장비

## 제공해야 하는 콘텐츠

-   숙련도 칭호
-   보스 러시
-   타임어택
-   퍼펙트 클리어 보상

------------------------------------------------------------------------

# Archetype 2 : 성장가 (Progression)

## 목표

계속 새로운 것을 해금하고 싶다.

## 좋아하는 요소

-   검술 해금
-   검파 해금
-   영구 성장
-   새로운 지역

## 제공해야 하는 콘텐츠

-   메타 성장
-   숨겨진 검술
-   새로운 시작 특성

------------------------------------------------------------------------

# Archetype 3 : 빌드 연구가 (Build Crafter)

## 목표

매 판 다른 조합을 만든다.

## 좋아하는 요소

-   로그라이크 선택
-   검술 조합
-   장비 시너지

## 제공해야 하는 콘텐츠

-   다양한 선택지
-   시너지 시스템
-   특수 조합

------------------------------------------------------------------------

# Archetype 4 : 수집가 (Collector)

## 목표

100% 달성

## 좋아하는 요소

-   도감
-   스킨
-   칭호
-   업적

## 제공해야 하는 콘텐츠

-   검술 도감
-   보스 도감
-   숨겨진 업적
-   희귀 꾸미기

------------------------------------------------------------------------

# Archetype 5 : 탐험가 (Explorer)

## 목표

새로운 것을 발견하는 재미

## 좋아하는 요소

-   숨겨진 길
-   비밀 보스
-   이벤트
-   특수 상호작용

## 제공해야 하는 콘텐츠

-   비밀 방
-   숨겨진 NPC
-   특수 이벤트

------------------------------------------------------------------------

# 콘텐츠 설계 규칙

새 기능을 추가하기 전에 반드시 확인한다.

□ 어떤 플레이어 유형을 위한 기능인가?

□ 기존 유형을 더 재미있게 만드는가?

□ 새로운 플레이 방식을 만드는가?

답이 없다면 구현하지 않는다.

------------------------------------------------------------------------

# MVP 우선 대상

출시 버전에서는 다음 세 유형에 집중한다.

1.  검술 마스터
2.  성장가
3.  빌드 연구가

수집가와 탐험가는 업데이트/DLC에서 확장한다.

------------------------------------------------------------------------

# 플레이어 여정

처음 30분 → 성장가

10시간 → 빌드 연구가

30시간 → 검술 마스터

100시간+ → 수집가 / 탐험가
