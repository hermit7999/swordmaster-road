# 1. 핵심 규칙

1. 보스는 새 규칙을 만들지 않고 기존 규칙(P03~05)을 조합한다.
2. 보스가 시험하는 검술의 교사 적이 같은 스테이지에 선행 등장한다 (P09 배치 규칙).
3. 패턴 수: 보스 3~6개, Phase로 조합 변화. 패턴 없이 HP만 높은 보스 금지.
4. 죽은 이유 설명 가능 — 전 패턴 Telegraph 규격(P04 §4) 준수.

# 4. Phase 규칙

- 전이 조건: HP 임계 (기본 60% / 30%) 또는 자세 붕괴 횟수.
- Phase 전이 시: 전이 연출(무적) → 패턴 풀 교체/추가 → 공격권 리셋(대치 복귀).
- Phase마다 새 패턴 최대 2개 추가 (한 번에 학습 가능한 양).
- B5만 3 Phase, 나머지 1~2 Phase.

# 5. 패턴 테이블 스키마 (boss_patterns.json)

```json
{
  "pattern_id": "ptn_veteran_thrust",
  "boss_id": "boss_veteran",
  "phase": 1,
  "zone": "mid",                     // far|mid|near — P05 거리 존
  "telegraph": { "type": "yellow", "ms": 800 },
  "sequence": [ { "move": "step_in", "ms": 300 }, { "attack": "thrust", "damage": 18, "poise": 20 } ],
  "opening_ms": 700,                 // 패턴 종료 후 빈틈 (반격 창)
  "cooldown_ms": 4000,
  "weight": 1.0,                     // 선택 가중치
  "tests_skill": "iaido",            // 이 패턴이 시험하는 플레이어 기술
  "version": 1
}
```
- 패턴 선택: 현재 거리 존 필터 → 쿨다운 필터 → weight 추첨. 같은 패턴 2연속 금지.
- 전 패턴에 `opening_ms`(빈틈) 필수 — "때릴 타이밍이 없는 보스" 금지.

# 6. B1 상세 예시 (구현 기준 — 나머지 보스는 동일 구조로 데이터만)

**boss_veteran — 낡은 검의 노병** (HP 300 / poise 300 / 1 Phase)

| 패턴 | 존 | 예고 | 내용 | 빈틈 | 시험 |
|---|---|---|---|---|---|
| 내려치기 | near | 노랑 800ms | 머리 위→수직 일격 | 900ms | 패링 or 측면 회피 |
| 찌르기 돌진 | mid/far | 노랑 900ms | 직선 돌진 | 700ms | 회피 후 반격 / 발도 카운터 |
| 대지 가르기 | near | 빨강 600ms | 전방 충격파 | 1100ms | 회피 강제 → 최대 반격 창 |

- 승리 경로: ① 패링 축적→붕괴→난무 ② 빈틈 반격 축적 ③ 혼합 — 3경로 모두 성립해야 Acceptance PASS.
- 목표 밸런스: 첫 도전 성공률 30~40%, 3회차 70%+ (Difficulty Bible S1 보정: 첫 보스는 55%/85%).

# 7. 보스 보상 (P07/P08 연결)

- 격파 시: 성장 선택 3택(레어 보장) + 골드 + 수련점. 첫 격파 보너스: 검술/오의 해금 이벤트.
- B1 첫 격파 = 오의 시스템 해금 서사 (연풍참).
