# 분 단위 설계

| 시간 | 플레이어가 하는 것 | 느껴야 하는 감정 | 시스템 |
|---|---|---|---|
| 0:00~0:30 | 타이틀 탭 → 밤길을 걷는다 (이동만) | 분위기 몰입, 조작 확인 | 튜토리얼 1 (UI §4) |
| 0:30~1:00 | 첫 잡병 — 화면을 긋는다 → **첫 베기** | "오? 베는 맛 좋은데" (첫 3초 검증) | 첫 베기 5요소 (PRS §7) |
| 1:00~2:30 | 잡병 3~4 연속 처치, 힌트 소멸 | "계속 베고 싶다" | 30초 루프 |
| 2:30~4:00 | 스승 이벤트 — **발도술 전수** (시범 궤적→따라 긋기→성공) | 첫 배움의 성취 | 관대 판정 |
| 4:00~5:30 | 방패 잡병 — 횡베기 막힘 → 내려베기 발견 | "아, 획마다 역할이 있구나" | 획 역할 학습 |

| 6:30~8:00 | 미니 결투 (M0급) — 배운 것 종합 | 첫 시험 통과 | 튜토리얼 졸업 |
| 8:00~9:00 | S1 진입 연출 + 첫 웨이브 (학살 템포) | 해방감·속도감 | 본편 시작 |
| 9:00~11:00 | 웨이브 + 들개 무리 → 포위 압박 | 소란스러운 재미 | 다수전 |
| 11:00~12:00 | **첫 성장 선택 3택** | "뭘 고르지?" 첫 고민 | 60초 루프 |
| 12:00~14:00 | 선택 효과 체감 웨이브 (달라진 플레이) | "플레이가 바뀌었다" | 성장 체감 검증 |
| 14:00~15:00 | 휴지 구간 (상점/휴식/이벤트) | 호흡, 소소한 선택 | 템포 파형 |
| 15:00~17:00 | 긴장 웨이브 + 정예(knight 계열 약체) | 난이도 상승 인지 | 2번째 성장 선택 |
| 17:00~19:00 | 미니보스 M1 산적 두목 | 진짜 첫 시험, 약간의 위기 | 간소 Duel |
| 19:00~20:00 | 보스 관문 앞 — 폭발 웨이브 마무리 | 고조 | 검기 첫 만충 유도 |
| 20:00~24:00 | **B1 노병 — 첫 Duel** (1~2회 시도 상정) | 긴장 → (아마도 1패) → "다음엔 되겠다" → 승리 | Duel 입문 |
| 24:00~25:00 | 보스 피니시 연출 + **오의 해금 이벤트 (연풍참)** | 최고 성취 + 새 장난감 | L4 보상 |
| 25:00~27:00 | S2 진입 — 연풍참 첫 성립 | "이게 오의구나!" | 오의 체감 |
| 27:00~30:00 | S2 웨이브 (archer 등장 — 새 과제) + 성장 선택 | 새 호기심 | 다음 루프 갈고리 |

# 검증 포인트 (실기 테스트 질문)

- 1:00 — 첫 베기에서 웃거나 감탄사가 나오는가
- 12:00 — 성장 선택 후 "달라졌다"를 스스로 말하는가
- 20:00대 — B1 1패 후 포기가 아니라 재도전을 누르는가
- 30:00 — 여기서 끊으라고 하면 "조금만 더"라고 하는가

# 규칙

- 30분 내 사망 최소화 설계 (S1 성공률 90%) — 단 B1에서 1패는 의도된 경험 (패배→학습→승리 사이클 각인).
- 30분 내 신규 시스템 등장 수 제한: 검술 4 + 회피 + 성장 선택 + Duel + 오의 = 충분. 검파·장비 심화는 30분 이후.
