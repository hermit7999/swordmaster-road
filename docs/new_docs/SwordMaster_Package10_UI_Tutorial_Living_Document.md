# 1. 절대 규칙 (프로토타입 버그4 교훈 — CLAUDE.md 규칙 8 승계)

1. **화면 전체가 검로 제스처 영역.** 모든 표시 요소(HUD·패널·토스트·연출)는 `pointer-events: none` 기본.
2. 실제 버튼만 예외적으로 인터랙션 허용 — 개수 최소화.
3. 전투 중 화면 중앙에 인터랙션 요소 배치 금지.

# 2. HUD 구성 (가로 고정)

| 위치 | 요소 |
|---|---|
| 좌상 | HP 바 + 기력 바 (겹층) |
| 좌상 아래 | 검기 게이지 (만충 시 금빛 펄스) |
| 우상 | 골드 · 스테이지 진행도 점 |
| 중앙 상단 | 콤보 카운터 (10+만 강조 표시) |
| 화면 가장자리 | Duel 공격권 색 (금빛/핏빛 비네트 — P05) |
| 좌하 (터치만) | 가상 스틱 존 힌트 (반투명, 첫 세션 후 소멸) |
| 우하 (터치만) | 검격 존 가이드 링 (은은, 상시) |
| 판정 피드백 | 적 위 등급 텍스트(퍼펙트~미스) + 데미지 — 0.5s 페이드 |

- 모바일: 좌반 터치=이동(스틱+상단 점프/회피 존), 우반=제스처 (프로토타입 트윈스틱 최종안 승계).
- PC: WASD 이동, 마우스 드래그=제스처, Space=회피.

# 3. 화면 목록 (총 9)

| 화면 | 핵심 요소 |
|---|---|
| 타이틀 | 시작/이어하기/수련의 방/도감/설정. 첫 진입 "탭하여 시작"(AudioContext 게이트) |
| 성장 3택 | 카드 3장: 이름/아이콘/효과/빌드 태그/희귀도. 스킵(골드) 버튼 |
| 상점 | 품목 5 + 재추첨. 보유 골드 상시 |
| 결과(사망/클리어) | 사인 or 클리어 타임 / 수련점 정산 / 미해금 미리보기 1개 / [재도전][수련의 방][타이틀] |
| 수련의 방 | 영구 성장 트리 + 승급 현황 + 검파 관리 |
| 도감 | 검술(숙련도 별)·적·보스. 미발견 실루엣 |
| 설정 | §5 |
| 일시정지 | 재개/설정/런 포기. Context=Pause (INP 규격) |
| 이벤트 | 텍스트 + 선택 2~3 |

# 4. 튜토리얼 (프롤로그 — "게임 전체가 튜토리얼"의 첫 5분)

- 별도 강의 화면 금지. 실전 시퀀스로: 
  1. 이동만 가능한 밤길 (조작 각인)
  2. 첫 잡병 — "긋기" 힌트 1회 (횡베기)
  3. 잡병 3 — 무힌트 반복
  4. 스승 이벤트 — 발도술 전수 (W→E 시범 궤적 표시, 성공까지 관대 판정)
  5. 방패 잡병 — 내려베기 필요성 체감
  6. 첫 회피 강제 상황 (빨강 예고 1회)
  7. M0급 미니 결투 — 배운 것 확인 → 본편
- 시범 궤적 오버레이는 `pointer-events: none` + 실제 판정 엔진 사용 (전용 판정 금지 — Integration Guide §6).
- 튜토리얼 완료 저장 — 재시작 시 스킵 가능.

# 5. 설정 항목

판정 난이도(관대/표준/도전 — 제스처 오차 스케일), 좌수 모드(미러), 가상키 크기, 이동 스틱 감도, 사운드(BGM/SFX 분리), 햅틱, 화면 흔들림 강도(접근성), 대화 속도, 언어(ko/en 준비).
