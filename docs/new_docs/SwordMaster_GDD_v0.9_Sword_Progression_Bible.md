# SwordMaster GDD v0.9

# Sword Progression Bible (초안)

## 목적

이 문서는 "레벨 시스템"을 정의하는 문서가 아니다.

이 문서는 플레이어가

> "내가 진짜 검객이 되어가고 있다."

라고 느끼게 만드는 설계 문서이다.

------------------------------------------------------------------------

# 핵심 철학

성장은 3개의 축으로 이루어진다.

1.  플레이어 실력
2.  캐릭터 영구 성장
3.  이번 플레이의 빌드 성장

세 가지가 동시에 성장해야 한다.

------------------------------------------------------------------------

# 성장 구조

## A. 영구 성장 (메타 진행)

죽어도 유지된다.

획득 가능한 요소

-   새로운 검파
-   새로운 검술 해금
-   패시브 슬롯
-   시작 특성
-   신규 지역
-   신규 보스 도전

목표

다음 플레이가 조금 더 다양해진다.

※ 단순 공격력 +10% 위주의 설계는 지양한다.

------------------------------------------------------------------------

## B. 이번 플레이 성장 (로그라이크)

보스를 쓰러뜨릴 때마다 3개의 선택지를 제시한다.

예시

-   발도술 강화
-   회전베기 강화
-   패링 특화
-   검기 습득
-   기력 증가
-   체력 증가

이번 플레이가 끝나면 초기화된다.

------------------------------------------------------------------------

## C. 플레이어 성장

실제로 손이 익는다.

예)

-   패턴을 읽는다.
-   제스처를 정확히 입력한다.
-   거리 조절을 익힌다.
-   회피 타이밍을 익힌다.
-   패링 성공률이 올라간다.

최종 목표는

"아이템이 아니라 내가 강해졌다."

라는 감정이다.

------------------------------------------------------------------------

# 검객 성장 단계 (초안)

1.  견습 검객
2.  초급 검사
3.  숙련 검사
4.  검객
5.  상급 검객
6.  검호
7.  대검호
8.  검왕
9.  검성
10. 검신

※ 각 단계마다 반드시 새로운 플레이 요소가 열린다.

절대로 숫자만 증가하지 않는다.

------------------------------------------------------------------------

# 단계별 해금 원칙

각 단계는 최소 하나 이상을 제공한다.

-   새로운 검술
-   기존 검술 진화
-   새로운 오의
-   검파 선택
-   패시브
-   장비 슬롯
-   새로운 빌드

------------------------------------------------------------------------

# 성장 검증 질문

각 성장은 아래 질문을 통과해야 한다.

□ 플레이 방식이 바뀌는가?

□ 새로운 선택이 생기는가?

□ 플레이어가 바로 체감하는가?

□ 실력을 대체하지 않는가?

□ 다음 플레이가 기대되는가?

------------------------------------------------------------------------

# 장기 플레이 목표

플레이어는

"레벨 100"

이 아니라

"모든 검파를 익히고 싶은 사람"

이 된다.

또는

"모든 검술을 마스터하고 싶은 사람"

이 된다.

# 다음 회의에서 반드시 결정할 것

1.  검술은 총 몇 개를 출시할 것인가?
2.  검파는 몇 개로 시작할 것인가?
3.  각 검파의 플레이 스타일은 무엇인가?
4.  영구 성장과 로그라이크 성장을 어디까지 구분할 것인가?

이 네 가지가 결정되면 실제 프로토타입 제작을 시작할 수 있다.
