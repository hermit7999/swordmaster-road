# 스토리 트리트먼트 v1.0 — 복수극 (ADR-013)

> 작성일: 2026-07-12 | 초안 — 인물명·대사는 사용자 검토 대상
> 원칙: 스토리는 감정 곡선(Psychology Bible)을 보조한다. 컷씬 최소, 대화 짧게, 스킵 가능. 인물 5명 이내.

---

# 1. 로그라인

몰락한 검술 명가의 마지막 생존자가, 가문을 멸문시킨 옛 대제자를 쫓아 검성봉을 오른다.
**복수의 길이 곧 수련의 길** — 스테이지를 오를수록 검이 완성된다.

# 2. 인물 5

| 인물 | 역할 | 시스템 연결 |
|---|---|---|
| **주인공** (무명) | 명가의 막내. 기초만 배운 채 멸문에서 도망침 | 승급 10단계 = 서사적 성장. 이름은 플레이어가 짓거나 무명 유지 |
| **스승 (백발노사)** | 가문의 늙은 검객. 은거지에서 주인공을 다시 세움 | 튜토리얼 교사(UI-07) + 수련의 방 화자 + 발도술 전수(2:30 이벤트) |
| **원수 (검성 흑월)** | 가문을 벤 옛 대제자. 검성 자리를 찬탈 | 최종보스 B5. 일섬 대결 연출 |
| **추격자 (그림자)** | 흑월이 보낸 암살 검사. 주인공을 반복 시험 | B4 그림자 검객 (미러 보스) + S4 shadow 적 서사 |
| **동행 (행상 소녀)** | 길 위의 정보상. 유일한 온기 | 상점 노드 얼굴 + 이벤트 노드 화자 |

# 3. 스테이지별 서사 (기존 테마에 배치)

| 스테이지 | 서사 비트 | 감정 |
|---|---|---|
| 프롤로그 | 멸문의 밤(회상, 이동만) → 은거지, 스승의 첫 가르침 | 상실 → 재기 |
| S1 산길 | 하산. 흑월의 영지 외곽 — 산적(사실은 패잔 가솔 사냥꾼) | 첫걸음 |
| S2 대나무 숲 | 옛 가문의 수련림. 무희(가문의 배신자 1)와 결투 | 배신의 확인 |
| S3 폐성 | 몰락한 본가. 수문장(가문의 배신자 2) — 가장 무거운 결투 | 귀향과 분노 |
| S4 설원 | 검성봉 초입. 그림자와의 재대결 — "너는 흑월과 같은 길을 걷는다"는 물음 | 복수의 회의 |
| S5 검성봉 | 흑월과 일섬 대결. 승리 후 선택 연출: 벤다/거둔다 (엔딩 분기 아님 — 여운 연출만) | 해소와 여운 |

- 보스 이름 확정: B1 노병=옛 가문 문지기였던 산적 두목의 상관 "낡은 검 갈퇴", B2 "홍련 무희", B3 "철벽 거암", B4 "그림자 무영", B5 "검성 흑월" (전부 가칭 — 사용자 검토).

# 4. 전달 방식 (1인 개발 규모 준수)

- 컷씬 없음. 스테이지 진입 1~2문장 나레이션 + 보스 전후 3~5줄 대화 (초상화+텍스트, 스킵/자동).
- 이벤트 노드(8종)에 세계관 조각 분산 — 읽는 사람만 깊어지는 구조.
- 총 대본 분량 목표: A4 6~8장. 대본 초안은 별도 작성 후 **사용자 검토 필수** (프로토타입 규칙 승계).

# 5. 반복 플레이 정합성 (로그라이크와 스토리의 충돌 해결)

- 서사 프레임: "수련의 길은 몇 번이고 다시 오른다" — 사망·재도전은 스승의 "다시."로 처리 (서사 내 설명).
- 스토리 대화는 첫 도달 시만 풀버전, 재방문 시 축약/스킵.
