# Gesture Recognition Acceptance Test Spec

> 목적
>
> 이 문서는 Gesture Recognition 구현이 완료되었는지를 객관적으로
> 판단하기 위한 합격 기준(Acceptance Criteria)을 정의한다.
>
> 구현 방법은 자유롭지만, 아래 테스트를 모두 통과해야 한다.

------------------------------------------------------------------------

# 1. 테스트 원칙

-   구현이 아니라 결과를 검증한다.
-   사람이 느끼는 품질과 수치를 함께 검증한다.
-   모든 AI가 동일한 테스트를 수행할 수 있어야 한다.

------------------------------------------------------------------------
