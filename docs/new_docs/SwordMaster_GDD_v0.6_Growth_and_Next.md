# 성장 철학 수정

초기에는 플레이어 실력 중심 게임을 목표로 했으나, 상용 게임으로서는
지나치게 진입 장벽이 높다.

따라서 다음과 같은 3축 성장 구조를 채택한다.

------------------------------------------------------------------------

# 1. 플레이어 성장 (Skill)

플레이어가 실제 컨트롤을 익힌다.

-   패턴 읽기
-   제스처 정확도
-   회피
-   거리 조절
-   검술 연계

최종적으로는

"내 컨트롤이 좋아져서 깼다."

라는 감정을 제공한다.

------------------------------------------------------------------------

# 2. 캐릭터 성장 (Permanent Progression)

실력이 부족한 플레이어도 조금씩 성장할 수 있도록 한다.

예시

-   기력 증가
-   체력 증가
-   패시브 해금
-   영구 강화
-   검술 숙련도

죽더라도 일부는 유지된다.

------------------------------------------------------------------------

# 3. 장비 성장 (Equipment)

장비는 숫자만 올리지 않는다.

장비는 플레이 스타일을 변화시킨다.

예)

-   발도술 사거리 증가
-   회전베기 범위 증가
-   패링 성공 시 검기 발생
-   회피 후 치명타 확률 증가

공격력 +1000 같은 단순 수치는 지양한다.

------------------------------------------------------------------------

# 장비 설계 원칙

장비는 실력을 대체하지 않는다.

장비는 실력을 더욱 빛나게 만든다.

예)

패링을 잘하는 사람은 패링 특화 장비를 착용했을 때 더 큰 효과를 얻는다.

------------------------------------------------------------------------

# 상용화 철학 보완

판매 대상

-   신규 지역
-   신규 검파
-   신규 보스
-   신규 플레이 스타일
-   꾸미기 요소

판매하지 않는 것

-   Pay to Win
-   능력치 판매
-   강한 검 판매
-   승리에 직결되는 과금

------------------------------------------------------------------------

# 현재까지 확정된 핵심 철학

-   손으로 검술을 직접 사용하는 게임
-   플레이어가 검객으로 성장하는 게임
-   실력이 가장 중요하다.
-   성장 시스템은 실력을 보완한다.
-   장비는 플레이 스타일을 확장한다.
