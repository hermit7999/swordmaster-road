# SwordMaster GDD v1.0

# Enemy Teaching Bible (초안)

## 핵심 철학

적은 플레이어를 죽이기 위한 존재가 아니다.

적은 플레이어에게 새로운 검술과 전투 원리를 '가르치는 스승'이다.

튜토리얼은 처음 5분만 존재하는 것이 아니라, 게임 전체가 튜토리얼이어야
한다.

------------------------------------------------------------------------

# 학습 구조

튜토리얼 ↓ 쉬운 적 ↓ 같은 기술을 요구하는 적 ↓ 응용 적 ↓ 미니 보스 ↓
보스(Duel Mode)

플레이어는 자연스럽게 기술을 익힌다.

------------------------------------------------------------------------

# 튜토리얼에서 배우는 것

## 1단계

-   이동
-   기본 베기

## 2단계

-   연속 베기
-   거리 유지

## 3단계

-   첫 검술(예: 발도술)

## 4단계

-   첫 회피

※ 튜토리얼에서는 실패해도 큰 패널티가 없다.

------------------------------------------------------------------------

# 이후는 적이 가르친다

## 방패병

학습 목표 - 패링 - 공격 타이밍

보스를 만나기 전에 반드시 등장한다.

------------------------------------------------------------------------

## 창병

학습 목표 - 거리 조절 - 접근 타이밍

------------------------------------------------------------------------

## 궁수

학습 목표 - 이동 - 회피

------------------------------------------------------------------------

## 쌍검 적

학습 목표 - 연속 공격 대응 - 리듬 유지

------------------------------------------------------------------------

## 검객

학습 목표 - 발도술 타이밍 - Duel 감각

------------------------------------------------------------------------

# 미니 보스

튜토리얼의 졸업 시험

지금까지 배운 기술 2\~3가지를 동시에 요구한다.

------------------------------------------------------------------------

# 보스(Duel Mode)

보스는 새로운 기술을 가르치지 않는다.

보스는 지금까지 배운 기술을 '시험'하는 존재이다.

------------------------------------------------------------------------

# 난이도 원칙

새로운 기술은 한 번에 하나만 요구한다.

절대로

패링 + 회피 + 발도술 + 검기

를 동시에 처음부터 요구하지 않는다.

------------------------------------------------------------------------

# 재미 검증

모든 적은 아래 질문에 답해야 한다.

1.  이 적은 무엇을 가르치는가?
2.  플레이어는 무엇을 연습하는가?
3.  이 적이 없으면 어떤 재미가 사라지는가?

답이 없다면 삭제한다.
