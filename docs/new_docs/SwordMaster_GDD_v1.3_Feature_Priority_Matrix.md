# 기능 추가 규칙

새 기능은 아래 6개를 통과해야 한다.

1.  재미가 증가하는가?
2.  검술 중심인가?
3.  1인 개발 가능한가?
4.  출시를 늦추지 않는가?
5.  반복 플레이를 높이는가?
6.  Steam 트레일러에서 매력적인가?

4개 미만이면 보류.

# MVP 체크리스트

출시 전 반드시 완료

□ 손맛 좋다. □ 일반전 재미있다. □ Duel Mode 재미있다. □ 성장 선택
재미있다. □ 보스를 다시 도전하고 싶다. □ 5분이 짧게 느껴진다.

------------------------------------------------------------------------

# 개발 원칙

절대로 기능 수를 자랑하지 않는다.

플레이어는

'기능이 많아서'

게임을 하는 것이 아니라

'재미있어서'

게임을 한다.
