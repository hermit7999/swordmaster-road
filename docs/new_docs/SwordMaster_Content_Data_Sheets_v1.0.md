# 5. 이벤트 노드 8종 (STG-04 입력 — 리스크 vs 리워드)

| 이벤트 | 선택 A | 선택 B |
|---|---|---|
| 낡은 사당 | 기도한다: 랜덤 패시브 | 부순다: 골드+200, 다음 웨이브 강화(저주) |
| 부러진 검비 | 검술 조각 읽기: 랜덤 검술 강화 | 지나친다: HP 20% 회복 |
| 행상 소녀 | 호위한다: 다음 상점 -30% | 거절한다: 즉시 골드+100 |
| 결투 신청자 | 받아준다: 간소 결투 → 승리 시 레어 선택지 | 피한다: 없음 |
| 독버섯 군락 | 채집: HP -15%, 전설 확률 +5%(이번 스테이지) | 우회: 없음 |
| 옛 가문의 문양 | 조사: 스토리 조각 + 수련점+1 | 외면: HP 10% 회복 |
| 떠도는 검령 | 대련: 기력 상한 +10 (이번 런) | 물러선다: 검기 +50% 충전 |
| 무너진 다리 | 뛰어넘는다: 시간 단축, HP -10% | 돌아간다: 추가 웨이브 1(골드 기회) |
