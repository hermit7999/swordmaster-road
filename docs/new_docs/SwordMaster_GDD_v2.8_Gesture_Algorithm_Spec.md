# 2. Point Buffer

Point

-   x
-   y
-   timestamp

권장

60Hz 이상

최소

30Hz

------------------------------------------------------------------------

# 3. Noise Filter

제거 대상

-   2px 이하 흔들림
-   동일 좌표 반복
-   매우 짧은 스파이크

목표

손 떨림 제거

------------------------------------------------------------------------

# 4. Normalize

입력을

0\~1 좌표계

로 변환한다.

기기 해상도 차이를 제거한다.

------------------------------------------------------------------------

# 5. Resample

모든 입력을

동일한 개수의 Point로 변환한다.

권장

64 Point

허용

32\~128

------------------------------------------------------------------------

# 6. Corner Detection

급격한 방향 변화만 유지한다.

Corner Angle

기본

35°

------------------------------------------------------------------------

# 7. Direction Vector

각 Segment를

8방향으로 양자화한다.

N NE E SE S SW W NW

예)

발도술

E → NE

------------------------------------------------------------------------

# 8. Pattern Score

총점 =

Direction 50%

Shape 20%

Length 10%

Speed 10%

Corner 10%

총점 85 이상

→ 성공

70\~84

→ 후보

69 이하

→ 실패

------------------------------------------------------------------------

# 9. 후보 충돌

동점이면

1.  Direction

3.  최근 성공률
4.  Skill Priority

순으로 결정
