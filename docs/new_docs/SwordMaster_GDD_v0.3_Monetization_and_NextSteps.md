# SwordMaster GDD v0.3 - 상용화 및 다음 설계

# 2026-07-12 추가 설계 기록

## 중요한 방향 전환

기존 RPG처럼 아이템을 파밍하고 강화하는 게임이 아니다.

이 게임의 핵심 수집 대상은

**아이템이 아니라 검술**이다.

------------------------------------------------------------------------

# 상용화 철학

## 하지 않을 것

-   Pay to Win
-   돈으로 더 강한 검 구매
-   돈으로 능력치 판매
-   돈으로 강한 검술 해금

검술은 플레이어가 직접 익혀야 한다.

실력이 핵심이다.

------------------------------------------------------------------------

# 장기 플레이 목표

플레이어는

더 좋은 검을 얻기 위해 플레이하는 것이 아니라

더 많은 검술을 익히고

새로운 검파를 배우며

다양한 빌드를 연구하기 위해 플레이한다.

------------------------------------------------------------------------

# 검술 도감

게임의 가장 중요한 수집 요소

-   검술 발견
-   검술 숙련
-   검술 마스터
-   오의 해금

Steam 업적과 연결

------------------------------------------------------------------------

# 로그라이크 성장

보스를 쓰러뜨릴 때마다

선택지 제공

예)

-   발도술 강화
-   회전베기 강화
-   검기 획득
-   패링 강화
-   새로운 검술 습득

매 플레이마다 다른 빌드 생성

------------------------------------------------------------------------

# 경제 시스템 원칙

플레이어는

실력으로 강해지고

콘텐츠에 돈을 쓴다.

강함을 돈으로 판매하지 않는다.
