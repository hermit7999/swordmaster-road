# Reward & Motivation Bible

> 목적
>
> 이 문서는 플레이어가 왜 계속 플레이하는지, 언제 보상을 주어야 가장 큰
> 만족감을 느끼는지를 정의한다.

------------------------------------------------------------------------

# 1. 보상 철학

보상은 단순히 아이템을 주는 행위가 아니다.

보상은

"내 선택이 옳았다."

라는 감정을 주는 것이다.

------------------------------------------------------------------------

# 2. 보상의 5단계

## Level 1 : 즉시 보상 (0\~3초)

행동

검을 긋는다.

즉시 받는 보상

-   타격감
-   효과음
-   적 반응
-   화면 연출

목표

베는 것 자체가 즐겁다.

------------------------------------------------------------------------

## Level 2 : 단기 보상 (30초)

행동

적 처치

보상

-   경험
-   작은 재화
-   회복
-   다음 전투 동기

목표

계속 적을 찾게 만든다.

------------------------------------------------------------------------

## Level 3 : 성장 보상 (1\~5분)

행동

정예/보스 처치

보상

3개의 선택지

예)

-   발도술 진화
-   회피 강화
-   새로운 검술
-   패시브

목표

플레이 방식이 바뀐다.

------------------------------------------------------------------------

## Level 4 : 세션 보상 (30\~40분)

행동

한 판 클리어

보상

-   영구 성장
-   검파 해금
-   새로운 보스
-   새로운 지역
-   신규 검술

목표

다음 판을 시작하게 만든다.

------------------------------------------------------------------------

## Level 5 : 장기 보상 (수십 시간)

행동

숙련

보상

-   검술 마스터
-   검신 칭호
-   숨겨진 검술
-   최고 난이도

목표

오랫동안 즐기게 만든다.

------------------------------------------------------------------------

# 3. 보상 설계 원칙

Rule 1

실력이 좋을수록 더 큰 보상을 받는다.

Rule 2

운보다 선택이 중요하다.

Rule 3

보상은 플레이 스타일을 바꾼다.

Rule 4

같은 보상이 반복되지 않는다.

------------------------------------------------------------------------

# 4. '한 판만 더'를 만드는 요소

플레이 종료 직전 최소 하나를 보여준다.

-   잠긴 검술
-   새로운 검파
-   미해금 보스
-   새로운 빌드 가능성
-   다음 지역 미리보기

플레이어가

"다음에는 저걸 해봐야지."

라고 생각해야 한다.

------------------------------------------------------------------------

# 5. 희귀 보상

낮은 확률의 특별 보상

예)

-   전설 검술
-   숨겨진 보스
-   특수 검파
-   특별 스킨

희귀하지만 게임을 깨뜨리지는 않는다.

------------------------------------------------------------------------

# 6. 보상 금지 사항

-   숫자만 큰 아이템
-   의미 없는 재화
-   선택 없는 성장
-   무조건 강해지는 장비
-   Pay to Win

------------------------------------------------------------------------

# 7. Motivation Loop

도전

↓

성공

↓

보상

↓

새로운 가능성

↓

호기심

↓

재도전

이 루프를 계속 유지한다.

------------------------------------------------------------------------

# 8. 플레이 테스트 질문

□ 다음 보상이 궁금한가?

□ 다른 빌드를 시도하고 싶은가?

□ 이번 선택을 후회하거나 만족하는가?

□ 게임을 끄기 전에 한 판 더 하고 싶은가?
