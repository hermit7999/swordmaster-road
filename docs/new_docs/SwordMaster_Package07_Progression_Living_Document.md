# 1. 목적/범위

포함: 런 성장(로그라이크 선택), 영구 성장(수련점), 승급 10단계, 검파 시스템, 검술 진화, 실패 후 유지 규칙, growth.json.
제외: 장비/드랍(P08), UI 화면(P10).

# 2. 런 성장 (이번 판 한정, 사망 시 초기화)

- 발생 시점: 웨이브 클리어 일부 / 정예·미니보스 / 보스 (P09 노드 규칙).
- **항상 3택** (Decision Bible §5). 선택지 구성 규칙: 서로 다른 빌드 축 — [공격형 / 기술형(검술·오의) / 생존형] 최소 2축 이상 섞임.
- 선택지 풀 (growth_options.json): 
  - 검술 강화 (예: 발도술 사거리 +25%)
  - 검술 진화 티어업 (Lv1→Lv5, Sword Language Bible §2)
  - 새 검술 습득 (미보유분)
  - 오의 강화 (예: 연풍참 화상 부여)
  - 검기 개조 (충전율/형태 변화)
  - 패시브 (기력 회복↑, 회피 후 가속 등)
  - 자원형 (최대 HP+, 기력+)
- 희귀도: 일반 70% / 레어 25% / 전설 5% (보스 후엔 레어 보장). 전설은 플레이 방식을 바꾸는 것만 (예: "패링 성공 시 검기 즉시 만충").
- 스킵 보상: 선택 대신 골드 (선택 강요 안 함).

# 3. 영구 성장 (수련점 — 사망 후 유지)

- 획득: 보스 격파, 도감 발견/마스터, 승급 시험, 업적.
- 사용처 (수련의 방):
  - 시작 특성 슬롯 (예: "발도류 입문 — 발도술 Lv2로 시작")
  - 패시브 슬롯 확장 (2→4)
  - 새 검술 영구 해금 (런 중 등장 풀에 추가)
  - 기본 능력 소폭 (최대 HP/기력 — 상한 +30%, "숫자만 성장" 방지 상한)
- **금지**: 위력 % 무한 성장. 영구 성장의 80%는 "선택지 다양화"여야 한다 (Progression Bible).

# 4. 승급 10단계 (검객 서열)

| 단계 | 칭호 | 승급 조건 (누적) | 해금 |
|---|---|---|---|
| 1 | 견습 검객 | 시작 | 기본 2검술 |
| 2 | 초급 검사 | 튜토리얼 완료 | 발도술 습득 자격 |
| 3 | 숙련 검사 | B1 격파 | 오의 시스템 |
| 4 | 검객 | 검술 5종 사용 숙련 | 검파 선택 |
| 5 | 상급 검객 | B2 격파 + 오의 2종 성립 | 패시브 슬롯+1 |
| 6 | 검호 | B3 격파 + 노데미지 웨이브 1회 | 검기 개조 |
| 7 | 대검호 | B4 격파 + 검술 도감 70% | 전설 선택지 확률↑ |
| 8 | 검왕 | B5 격파 (엔딩) | 도전 난이도 |
| 9 | 검성 | 도전 난이도 B5 격파 | 일섬 상시 해금 |
| 10 | 검신 | 도감 100% + 검성 승급 시험 | 칭호·비전 스킨 (성능 없음) |

- 승급 시험: 4→5부터는 조건 충족 시 "승급 결투" 이벤트 (전용 상대, 프로토타입 승급 씬 개념 재사용).
- 각 단계 = 반드시 새 플레이 요소 해금 (숫자만 증가 금지 검증).

# 5. 검파 (승급 4단계에서 선택, 이후 수련점으로 추가 해금)

| 검파 | 정체성 | 시작 보너스 | 전용 선택지 예 |
|---|---|---|---|
| 발도류 | 카운터·일격 | 발도술 Lv2 시작 | "일섬 쿨다운 제거", "카운터 배율 +50%" |
| 정검류 | 정면·수비 반격 | 패링 유효창 +15% | "연쇄 패링 보너스 2배", "가드브레이크 강화" |
| 쾌검류 | 속도·연계 | 콤보 유지 +25% | "사선베기 3연", "콤보 15+ 시 기력 무한" |

- 검파는 검술을 배타 소유하지 않음 — 강화 방향만 (Sword Language Bible §5).
- 런 시작 시 검파 선택 가능 (해금분 중).

# 6. 실패(사망) 후 유지 규칙

| 유지 | 초기화 |
|---|---|
| 수련점, 영구 해금, 승급, 도감, 업적, 옵션 | 런 빌드(선택지), 골드, 검기 게이지, 스테이지 진행 |

# 9. Acceptance Test

□ 3택이 항상 2축 이상 혼합 / 이미 만렙 선택지 미등장 / 중복 0
□ 진화 Lv1→5 각 단계 효과 실측 (플레이 방식 변화 검증 질문 5종)

□ 검파 보너스·전용 풀 동작
□ 사망 후 유지/초기화 항목 표 §6과 완전 일치
□ 사망 화면 3요소(사인/수련점/미리보기) 데이터 정확
