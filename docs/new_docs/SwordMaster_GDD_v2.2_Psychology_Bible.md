# SwordMaster GDD v2.2

# Psychology Bible

> 목적
>
> 이 문서는 플레이어가 어떤 감정을 어떤 순서로 느껴야 하는지를 정의한다.
> 게임 시스템은 감정을 만들기 위한 도구이며, 감정이 목표이다.

------------------------------------------------------------------------

# 1. 핵심 철학

플레이어는 기능을 기억하지 않는다.

플레이어는 감정을 기억한다.

따라서 모든 시스템은 아래 감정 곡선을 강화해야 한다.

------------------------------------------------------------------------

# 2. 핵심 감정 곡선

호기심

↓

성공

↓

성장

↓

긴장

↓

실패

↓

학습

↓

재도전

↓

성취

↓

새로운 기대

이 흐름은 모든 스테이지에서 반복된다.

------------------------------------------------------------------------

# 3. 첫 30분 감정 설계

0\~3분

"오, 베는 맛 좋다."

3\~10분

"새로운 검술이 재미있다."

10\~20분

"조금 어렵지만 할 만하다."

20\~30분

"보스를 잡았다!"

30분 이후

"다른 빌드도 해보고 싶다."

------------------------------------------------------------------------

# 4. 실패 심리

실패는 좌절이 아니라 학습이어야 한다.

플레이어는 패배 후 다음 중 하나를 생각해야 한다.

-   타이밍이 늦었다.
-   다른 검술을 써볼까?
-   이번에는 회피를 먼저 해보자.

절대로

"운이 없었다."

"게임이 이상하다."

를 느끼게 하면 안 된다.

------------------------------------------------------------------------

# 5. 성취감 설계

큰 성취는 반드시 어려운 도전을 극복한 뒤에 온다.

예시

-   첫 패링 성공
-   첫 노데미지 승리
-   첫 Duel Mode 클리어
-   첫 검파 해금
-   첫 검성 승급

------------------------------------------------------------------------

# 6. 몰입(Flow)

몰입은

난이도와 실력이 균형을 이룰 때 발생한다.

너무 쉽다 → 지루함

너무 어렵다 → 좌절

적당히 어렵다 → 몰입

------------------------------------------------------------------------

# 7. 놀라움

게임은 주기적으로 예상 밖의 경험을 제공해야 한다.

예시

-   숨겨진 검술
-   비밀 보스
-   특수 이벤트
-   희귀 빌드 시너지

------------------------------------------------------------------------

# 8. 재도전 욕구

게임 종료 직전

최소 하나의 미완성 목표를 보여준다.

-   아직 배우지 못한 검술
-   잠긴 검파
-   새로운 난이도
-   숨겨진 보스

플레이어는

"다음에는 저걸 해보자."

라고 생각해야 한다.

------------------------------------------------------------------------

# 9. 감정 체크리스트

모든 신규 기능은 아래를 만족해야 한다.

□ 웃는 순간이 있는가?

□ 긴장되는 순간이 있는가?

□ 실패가 억울하지 않은가?

□ 성공이 특별한가?

□ 한 판 더 하고 싶은가?
