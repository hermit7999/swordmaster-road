# 전투 재설계안 v2 — "공격권을 빼앗는 검객" (2026-07-11)

상태: **사용자 승인 대기** — 승인 후 기존 전투를 이 설계로 교체하는 스파이크 착수.
근거: 사용자 재검토 문서 2편. §7 합의 기준(완성 빌드 미달 시 컨셉 재설계) 발동.

---

## 0. 무엇이 실패했고 무엇을 유지하는가

**실패 (폐기)**: "적의 행동을 보고 정답 방향을 입력하는" 반응형 구조 전체.
세 차례 수정(3층 루프→연출→자세)에도 플레이어 경험은 "시험 응시"에서
벗어나지 못함. 원인 = 주도권이 항상 적에게 있고, 행동이 사실상 하나이며,
정보가 적이 아닌 UI에 있었기 때문.

**유지 (검증된 자산)**: 판정 엔진(획/등급), 검술·아츠·마나 공식, 숙련·경지,
검로/검결 병행, 종이인형 연출 기술, 아트, 스토리. 긋기는 **조작법으로만**
남는다 — 게임은 긋기가 아니라 검객의 싸움이다.

## 1. 새 중심 메커니즘 — 공격권(攻擊權)

전투는 항상 셋 중 한 상태다. 화면 연출로 상태가 즉시 구분된다(전용 UI 최소).

| 상태 | 화면 | 플레이어가 할 것 |
|---|---|---|
| **내 공세** | 적이 밀리며 방어 자세, 미세 슬로우, 화면 가장자리 금빛 | 마음껏 벤다 — 모든 획이 꽂히고 콤보 배율 상승, 검기 충전 |
| **대치** | 서로 노려봄 | 선택: 먼저 친다(가드 깎기) / 기다린다(적 반격 유도) |
| **적 공세** | 적이 압박, 화면 가장자리 핏빛 | 회피(플릭) 또는 패링 — 성공하면 공격권 강탈 |

**공격권을 얻는 법**: ① 패링 성공 ② 회피 직후 반격 ③ 가드 브레이크(막힌
방향도 계속 치면 가드 게이지가 부서짐 — 공격은 항상 전진) ④ 일반전은
시작부터 내 것(기습).
**잃는 법**: 미스 획, 공세 시간 소진, 적의 빨강 기술.

→ 문서의 요구 흐름 그대로: 내가 공격 → 적 방어 → 적 반격 → 회피/패링 →
공격권 획득 → 콤보 → 자세 붕괴 → 피니시.

## 2. 행동 어휘 — "정답 하나"에서 "선택지 다섯"으로

같은 긋기 입력으로 상황마다 다른 수가 성립한다:

1. **평타 획** — 빠른 견제, 가드 깎기
2. **강공격** — 길게 눌러 긋기(차지): 느리지만 가드 관통, 빈틈 크면 최선
3. **검기** — 공세 중 충전된 게이지 소모, 화면을 관통하는 참격(다수 절단)
4. **플릭 회피** — 좌/우 휙: 빨강 기술 회피, 직후 반격 보너스
5. **패링** — 노랑 기술에 카운터 획: 최고 리스크·최고 보상(즉시 공격권+그로기)

매 순간의 질문이 "무슨 방향?"이 아니라 **"지금 뭘 할까?"**가 된다.

## 3. 공격의 결과가 전장을 바꾼다

정타마다 최소 하나가 변한다: 적이 크게 밀림 / 가드 게이지 파손(방패·무기가
눈에 보이게 부서짐) / 다수 관통 절단 / 검기 충전 / 자세 붕괴 접근.
데미지 숫자만 바뀌는 타격은 없앤다.

## 4. 세 전투는 다른 게임이다

| | 일반전 (8~10초) | 정예전 (20~40초) | 보스전 (1~3분) |
|---|---|---|---|
| 정체 | **압도** — 학살 쾌감 | **쟁탈** — 공격권 루프의 본체 | **결투** — 심리전 |
| 흐름 | 시작부터 공격권 내 것. 다수 적 웨이브, 한 획 다수 절단, 검기 마무리 | 대치→가드 깎기→반격 유도→패링→공세→붕괴→피니시 | 공격권 쟁탈 + 페인트(노랑인 척 빨강), 페이즈 변화, 최후 일섬 |
| 위험 | 낮음 (기분 좋으라고 있는 전투) | 중간 | 높음 |

## 5. UI 원칙 — 적을 본다

방향 지시 UI(나침반·화살표·"이 획을 그어라") 전면 폐지. 모든 전투 정보는
적의 몸에: 가드 방향=무기 든 포즈, 공격 종류=몸의 발광색(노랑/빨강),
빈틈=자세가 무너진 실루엣. 게이지는 공격권/검기 2개만, 최소 크기.

## 6. 구현 계획 (승인 후)

1. **스파이크 1 — 정예전 1종(낭인검객)**: 공격권 FSM + 행동 5종 + 가드
   브레이크. §4 정예전 흐름이 한 판에 성립하는지 검증 ← **여기서 1차 판정**
2. 스파이크 2 — 일반전 웨이브(기습 학살 + 검기 피니시, 8~10초)
3. 스파이크 3 — 보스(페인트·페이즈·일섬)
4. 통과 시 본 게임 이식, T2-08/09 재개

기존 combat 모듈은 보존한 채 신규 momentum FSM으로 병행 개발(스파이크 모드).
판정 엔진·데이터 구조 재사용, 신규 수치는 전부 balance.momentum.

## 7. 판정 질문 (갱신)

"한 판 더 하고 싶은가"에 하나 추가:
**"내가 싸움을 지배하고 있다고 느끼는가."**

## 8. 정직한 경계선

이 재설계는 "긋기 위에서 검객의 경험을 만들 수 있다"는 마지막 대형 가설이다.
스파이크 1(정예전 쟁탈 루프)에서도 지배감이 안 나오면, 그때는 긋기를 기본
조작에서 내리고(자동 전투+긋기는 필살기 전용 등) 게임 장르 차원의 전환을
논의한다 — 그 결정은 M2 데모 범위 재정의를 동반한다.
