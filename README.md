# 소드마스터의 길 (Path of the Swordmaster)

획(劃)을 그어 검을 휘두르는 제스처 검술 액션 RPG. 모바일+PC 웹, 단일 코드베이스, 가로 고정.

## 실행

정적 파일이라 서버가 필요 없다. `index.html`을 브라우저로 열거나:

```
python -m http.server 5178 --directory .
# http://localhost:5178/index.html
```

가로 화면 권장. 세로면 회전 안내가 뜬다.

## 입력 두 방식 (동시 활성)

- **검로(劍路)** — 화면을 터치/마우스로 **긋는다**. 정확도=궤적 정밀도.
- **검결(劍訣)** — 방향키/가상패드 커맨드(격투게임식). 정확도=리듬/타이밍(리듬판정은 T0-05 예정).

### M0 획 3종 (우수검류)

| 획 | 검로(궤적) | 검결(커맨드) |
|---|---|---|
| 횡베기(좌→우) `h_lr` | 좌→우 직선 | `←→` (순차) |
| 사선베기(↘) `diag_dr` | ↘ 직선 | `↓+→` (동시) / 패드 `↘` |
| 내려찍기 `v_down` | 상→하 직선 | `↑↓` (순차) |

- 소드 아츠 **연풍참** = 횡베기 → 사선(↘), 1.5s 창. 성립 시 마나 = 15 × (2 − 평균정확도/100).
- 유파 토글(우수/좌수검류): 좌수는 좌우 미러 + ↑↓ 클러스터 좌측.
- `자가진단` 버튼: 판정/커맨드 엔진 회귀 테스트를 화면·콘솔에 출력.

## 구조 (작업 규칙)

- M0는 **단일 `index.html`** (프레임워크 금지). M1부터 Phaser 3 + TS + Vite 전환.
- 판정 로직은 **순수 함수**(DOM 의존 금지) — `judgeStroke()`, `recognizeCommand()`.
- 수치는 `BALANCE / STROKE_TEMPLATES / TECHNIQUES / STYLES` **DATA 블록**에만.
- 판정 이벤트 규격 고정: `{strokeId, accuracy, grade, inputMode, timestamp}`.
